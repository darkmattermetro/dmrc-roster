# DMRC Line 7 Roster Master - Web App

A simple HTML/JS web app for DMRC Line 7 metro crew to search duty schedules.

## Quick Deploy to Vercel

### Option 1: Drag & Drop (Easiest)

1. Go to: https://vercel.com/drop
2. Drag the `vercel-web` folder to the page
3. Done! Your app is live!

### Option 2: Via GitHub

1. Create a GitHub repo
2. Upload files from `vercel-web` folder
3. Go to vercel.com → Import Project
4. Deploy!

## Local Testing

Simply open `vercel-web/index.html` in a browser.

## Features

- Duty Search by number
- Day Type selection (Weekday/Saturday/Sunday/Special)
- KM Calculation
- Rake Analysis (Reliever Points)
- Mobile responsive
- PWA ready

## Files

```
vercel-web/
├── index.html     # Main app (copy this to root)
└── vercel.json   # Vercel config
```

## Customize

Edit the `kmData` and `dutiesData` objects in `index.html` to add your data.
